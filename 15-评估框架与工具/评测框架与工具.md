## 快速评测命令

### lm-eval-harness
```bash
# 安装
pip install lm-eval

# 评测 MMLU + GSM8K
lm_eval --model hf \
  --model_args pretrained=meta-llama/Llama-3-8B \
  --tasks mmlu,hellaswag,gsm8k \
  --num_fewshot 5

# 评测 GPQA Diamond
lm_eval --model hf \
  --model_args pretrained=meta-llama/Llama-3-8B \
  --tasks gpqa_diamond \
  --num_fewshot 0
```

### DeepEval
```python
from deepeval.benchmarks import MMLU, GSM8K, HumanEval

# MMLU
mmlu = MMLU(tasks=[...], n_shots=3)
mmlu.evaluate(model=my_model)
print(f"MMLU Score: {mmlu.overall_score}")

# GSM8K
gsm8k = GSM8K(n_problems=100, n_shots=8, enable_cot=True)
gsm8k.evaluate(model=my_model)
print(f"GSM8K Score: {gsm8k.overall_score}")
```

### rLLM
```bash
# 安装
pip install rllm

# 列出可用数据集
rllm dataset list --all

# 评测 GSM8K
rllm eval gsm8k

# 评测 MMLU-Pro
rllm eval mmlu_pro
```
