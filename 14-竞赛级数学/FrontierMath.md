# FrontierMath — 前沿数学评测

## 基本信息

| 属性 | 值 |
|------|-----|
| 全称 | FrontierMath: A Benchmark for Evaluating Advanced Mathematical Reasoning |
| 创建者 | Epoch AI |
| 发布年份 | 2025 |
| 题目数量 | 数百道 |
| 题型 | 自由格式数学题 |
| 数据集链接 | https://huggingface.co/datasets (搜索 FrontierMath) |
| 状态 | ✅ 极具区分度（top 模型 < 10%） |

## 评测维度

- 由专业数学家精心设计
- 覆盖代数、分析、数论、组合、几何等高级数学领域
- 每道题需要深层数学推理
- 答案为可验证的数值或简单表达式

## 特点

1. **极高难度**: 远超 AIME 和 MATH
2. **专业数学**: 需要研究生以上水平数学知识
3. **防污染**: 新设计的题目
4. **私有评测**: 防止训练数据泄露

## 各模型表现（2025年底）

| 模型 | FrontierMath |
|------|-------------|
| 前沿模型 | < 10% |
| 人类数学家 | ~80%+ |

## 为何 FrontierMath 重要

1. **测量数学推理上限**: 唯一能测量 AI 数学推理极限的 benchmark
2. **未饱和**: 当前最高模型 < 10%
3. **专业设计**: 由数学家设计，非自动化生成
4. **推动数学 AI**: 推动数学推理能力突破
